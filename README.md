# Truthdb/apt

Signed apt repository for [TruthDB](https://github.com/Truthdb/truthdb), served via GitHub Pages at <https://truthdb.github.io/apt/>.

Install instructions live in [`index.html`](index.html). User-facing copy of the same content is at `https://truthdb.github.io/apt/`.

## Layout

- `conf/distributions` — reprepro suite definitions (bookworm, trixie, jammy, noble). `${KEY_ID}` is substituted by the publish workflow with the `APT_GPG_KEY_ID` secret before reprepro runs.
- `pubkey.asc` — public signing key. Users fetch this and place it under `/etc/apt/keyrings/`.
- `.github/workflows/publish.yml` — triggered by a `repository_dispatch` event of type `new-debs` sent from `Truthdb/truthdb`'s release workflow. Fetches the tagged release's `.deb` assets, runs `reprepro includedeb` into each suite, commits `dists/`, `pool/`, and `db/` back to this repo.
- `dists/`, `pool/`, `db/` — generated by reprepro and committed by the workflow. Do not edit by hand.

## Required secrets

| Secret | What it is |
|---|---|
| `APT_GPG_PRIVATE_KEY` | ASCII-armored private key (output of `gpg --armor --export-secret-keys <KEY_ID>`) |
| `APT_GPG_PASSPHRASE` | Passphrase for the private key |
| `APT_GPG_KEY_ID` | Long-form key ID, e.g. the hex string after `rsa4096/` in `gpg --list-secret-keys --keyid-format=long` |
